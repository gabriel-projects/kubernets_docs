# Glossário

## Objetivo
Centralizar termos importantes de Kubernetes e AKS para revisão rápida.

## Termos
- **Cluster**: conjunto de nós que executa workloads Kubernetes.
- **Node**: máquina (VM) que executa pods.
- **Pod**: menor unidade de execução no Kubernetes.
- **Deployment**: controlador de rollout/rollback para pods stateless.
- **Service**: endpoint estável para acesso a pods.
- **Ingress**: roteamento HTTP/HTTPS para serviços.
- **ConfigMap**: configuração não sensível.
- **Secret**: dado sensível (token, senha, chave).
- **Namespace**: isolamento lógico de recursos.
- **Node Pool (AKS)**: grupo de nós com mesma configuração.
